# Configuration

## Leaderboard

The **First Leaderboard** action exposes settings for page size, header, entry format, separator, empty-state text, and invalid-page text.

Supported placeholders include `{page}`, `{totalPages}`, `{rank}`, `{user}`, `{count}`, and `{requester}` where applicable.

## First Me

The **First Me** action exposes message templates and role-based controls for looking up other users. Available roles include broadcaster, moderator, VIP, subscriber, and follower.

## Storage paths

Source defaults use:

- Parent folder: `Omanges Extensions`
- Data folder: `First`
- Leaderboard file: `FirstLeaderboard.json`
- History file: `FirstHistory.json`
